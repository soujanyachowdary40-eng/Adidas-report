# Adidas Sales Analysis Dashboard

## Project Overview

This project analyzes Adidas sales data using Power BI. The dashboard provides insights into sales performance, profit trends, product categories, retailer performance, and regional sales distribution.

## Objectives

* Analyze Adidas sales performance.
* Identify top-selling products.
* Compare sales across regions and retailers.
* Track profit and revenue trends.
* Support data-driven business decisions.

## Tools Used

* Power BI
* Microsoft Excel
* Data Visualization
* Data Analysis

## Dashboard Features

* Total Sales Analysis
* Profit Analysis
* Sales by Region
* Sales by Product Category
* Retailer Performance Analysis
* Monthly Sales Trends
* Interactive Filters and Slicers

## Key Insights

* Identified top-performing regions and products.
* Analyzed retailer contribution to overall sales.
* Observed seasonal sales patterns.
* Evaluated profit margins across categories.

## Dataset

The dataset contains Adidas sales information including:

* Product Name
* Sales Amount
* Profit
* Region
* Retailer
* Date of Sale

## Conclusion

The Adidas Sales Dashboard helps businesses understand sales trends, improve performance, and make informed strategic decisions through interactive visualizations.
